# Prompts para recrear VendoYo Landing

Esta carpeta contiene los prompts utilizados para crear la landing page de VendoYo con un agente de IA.

## Orden de ejecución

1. `01-setup.md` - Configuración inicial del proyecto
2. `02-design-system.md` - Sistema de diseño y tokens CSS
3. `03-layout-seo.md` - Layout base y SEO
4. `04-header.md` - Header con navegación
5. `05-hero.md` - Sección Hero
6. `06-features.md` - Sección de características
7. `07-pricing.md` - Sección de precios
8. `08-testimonials.md` - Testimonios
9. `09-faq.md` - Preguntas frecuentes
10. `10-app-showcase.md` - Showcase de la app
11. `11-footer.md` - Footer
12. `12-legal-pages.md` - Páginas legales y 404
13. `13-islands.md` - Interactividad (islands)
14. `14-mobile-mockups.md` - Mockups de pantallas móviles
15. `15-finalization.md` - Optimización y build final

## Uso

Cada archivo contiene el prompt que se puede usar con un agente de IA (Claude, GPT, etc.) para recrear esa parte específica del proyecto.

Los prompts asumen que se ejecutan en orden y que el proyecto base ya existe.
