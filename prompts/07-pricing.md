# 07 - Pricing Section

## Prompt

```
Crea la sección de precios para VendoYo.

Ubicación: src/components/Pricing.astro

Enfoque B2C:
- Destacar tarifa plana vs comisiones tradicionales (3-5%)
- Enfatizar el ahorro para el propietario
- Mensaje claro y directo

Estructura:
- Header con título impactante
- Comparativa visual: "Tarifa plana" vs "Comisión tradicional"
- Card principal con el precio y servicios incluidos
- Lista de beneficios con iconos check
- CTA de descarga
- Nota al pie "*sin IVA" donde aplique

Contenido sugerido:
- Precio: "Desde 2.950€" (o el que corresponda)
- Ahorro ejemplo: "Ahorra más de 10.000€ en una venta de 300.000€"
- Servicios incluidos:
  - Fotografía profesional
  - Publicación en portales
  - Gestión de visitas
  - Asesoría legal
  - Acompañamiento hasta la firma

Diseño:
- Fondo que destaque (puede ser gradiente suave)
- Card con sombra y hover effect
- Iconos para cada beneficio
- Responsive mobile-first

Animaciones (opcional):
- Fade-up al entrar en viewport
- Counter animado para el ahorro (si se implementa)
```

## Resultado esperado

- Sección de precios clara
- Propuesta de valor evidente
- CTA prominente
